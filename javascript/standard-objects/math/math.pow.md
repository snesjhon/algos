# Math.pow\(\)

The `Math.pow()` function returns the `base` to the `exponent` power. That is: 

$$
base^{exponent}
$$

Using this we calculate the base to the exponent solution.

{% tabs %}
{% tab title="Example" %}
```javascript
Math.pow(7,3) // 343
```

This would be equal to `7 X 7 X 7`
